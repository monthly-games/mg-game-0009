## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 카드/덱 구조 정착 |
| 7~10주 | 퍼즐 난이도/보상 조정 |
| 11~12주 | 첫 시즌 카드 준비 |

**핵심 과제:**
- 카드 수집 시스템 안정화
- 덱 구성의 재미 밸런스
- 퍼즐 난이도 최적화

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 시즌 카드/테마 덱 1차 추가 |
| 5개월 | 주간 덱 제한 이벤트 시작 |
| 6개월 | 시즌 카드/테마 덱 2차 추가 |
| 7개월 | 속성/코스트 제한 이벤트 |
| 8개월 | 시즌 카드 3차 + 여름 이벤트 |

**핵심 과제:**
- 시즌 카드 사이클 정착
- 덱 제한 이벤트 효과 검증
- 카드 풀 확장

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 0016(덱빌딩 히어로즈) 카드 풀 연계 설계 |
| 10개월 | 상위 유저용 빌드 가이드 시스템 |
| 11개월 | 카드 일러 한정 이벤트 |
| 12개월 | 1주년 이벤트 + 콜라보 카드 |

**핵심 과제:**
- 0016 연계 카드 풀 설계
- 가이드/추천 시스템
- 일러/콜라보 이벤트
