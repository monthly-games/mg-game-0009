## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **성장 쾌감** | 먹이를 먹고 점점 길어지는 뱀 | ★★★★★ |
| **공간 관리** | 자신의 몸을 피하며 이동하는 전략 | ★★★★☆ |
| **기록 경쟁** | 최고 점수 달성 | ★★★★☆ |
| **단순함** | 직관적이고 쉬운 조작 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **먹이 획득**: 먹이를 먹을 때마다 점수와 길이 증가
- **위기 탈출**: 좁은 공간에서 빠져나올 때의 긴장감 해소
- **신기록**: 최고 길이/점수 갱신
- **보너스 먹이**: 특수 먹이 획득

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 점점 길어지는 몸 관리
├── 속도 증가 (점수에 따라)
├── 하드 모드 (장애물 추가)
├── 타임 어택 모드
└── 무한 모드
```

### 1.4 사회적 요소 (Social)
- **리더보드**: 글로벌/친구 점수 랭킹
- **기록 공유**: SNS 점수 공유
- **일일 챌린지**: 동일 조건 경쟁
- **고스트 경쟁**: 친구 기록과 비교

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (30초~5분)
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [이동] → [먹이 탐색] → [먹이 획득]              │
│     ↑                        ↓                   │
│     └──── [게임 오버] ← [충돌 체크] ←┘           │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **이동** (지속): 스와이프로 방향 전환
2. **먹이 탐색** (지속): 먹이 위치 확인
3. **먹이 획득** (즉시): 먹이 먹고 점수/길이 증가
4. **충돌 체크** (지속): 벽/자기 몸 충돌 확인
5. **게임 오버** (5초): 충돌 시 게임 종료, 결과 확인

### 2.2 세션 루프 (5~15분)
```
[접속] → [일일 보상] → [모드 선택] → [게임] → [게임 오버]
                                                    ↓
                                   [재도전] ← [보상 확인]
```

**세션 구성:**
- 일일 보상: 10초
- 게임 3~5회: 3~12분
- 업그레이드/스킨 확인: 2~3분

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [점수 달성] → [골드 획득] → [스킨 해금]              │
│         ↓                              ↓             │
│  [새 모드 해금]                  [업그레이드 강화]     │
│         ↓                              ↓             │
│  [더 높은 점수 도전] ←──────────────────┘            │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 메뉴] ────→ [스킨 선택]
    │             [상점]
    │             [리더보드]
    │             [설정]
    ↓
[모드 선택] → [게임 화면] → [게임 플레이]
                               ↓
                          [게임 오버]
                               ↓
                          [결과/재시작]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **리워드 광고** | 50% | 보상 2배, 부활, 파워업 |
| **스킨** | 25% | 뱀 스킨, 먹이 스킨 |
| **광고 제거** | 15% | 일회성 구매 |
| **인터스티셜** | 10% | 게임 오버 후 광고 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 게임 모드 플레이 가능
├── 기본 스킨 사용
├── 광고 시청으로 보상 2배
└── 일일 무료 골드 보너스

과금 유저 혜택:
├── 광고 제거 ($1.99)
├── 스킨 팩 ($0.99~$2.99)
├── 골드 패키지 ($0.99~$4.99)
├── 스타터 팩 ($2.99)
└── 전체 스킨 해금 ($5.99)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 골드 (500) | $0.99 | 스킨 1개 |
| 중량 골드 (2500) | $2.99 | 20% 보너스 |
| 대량 골드 (6000) | $4.99 | 40% 보너스 |
| 광고 제거 | $1.99 | 영구 적용 |
| 스킨 팩 | $0.99~2.99 | 개별 스킨 |
| 올 스킨 팩 | $5.99 | 모든 스킨 해금 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 출석 체크 (7일 사이클)
- **일일 퀘스트**: 점수 달성, 먹이 개수
- **주간 챌린지**: 특수 조건 달성
- **시즌 스킨**: 한정 기간 스킨

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
속도
  ▲
  │                              ╭──────
  │                         ╭────╯
  │                    ╭────╯
  │               ╭────╯
  │          ╭────╯
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 점수
    0   10   20   30   40   50   60
```

**난이도 공식:**
```
이동_속도 = 기본_속도 × (1 - 속도_업그레이드 × 0.05)
게임_속도 = 기본_속도 + 점수 × 0.5
최대_속도 = 기본_속도 × 2.5
```

### 4.2 콘텐츠 진행
| 모드 | 해금 조건 | 특징 |
|------|----------|------|
| 노멀 | 시작 | 기본 게임, 벽 충돌 |
| 하드 | 점수 30 | 장애물 추가 |
| 타임 어택 | 점수 50 | 60초 제한 시간 |
| 무벽 | 점수 80 | 벽 통과 (반대쪽 출현) |
| 무한 | 점수 100 | 점수 제한 없음 |

### 4.3 밸런싱 공식
```dart
// 점수 계산
int calculateScore(int foodEaten, GameMode mode) {
  int baseScore = foodEaten;
  double modeMultiplier = getModeMultiplier(mode);
  double upgradeBonus = getScoreUpgradeBonus();
  return (baseScore * modeMultiplier * (1 + upgradeBonus)).toInt();
}

// 골드 보상
int goldReward(int score, GameMode mode) {
  return score * 2 + getModeBonusGold(mode);
}

// 업그레이드 효과
double getSpeedReduction(int level) {
  return level * 0.05; // 레벨당 5% 감소 (느려짐 = 쉬움)
}

double getScoreMultiplier(int level) {
  return 1 + level * 0.1; // 레벨당 10% 점수 증가
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 점수 골드 | 게임 종료 | 점수×2 |
| 모드 보너스 | 하드/타임어택 | +50% |
| 일일 보너스 | 출석 | 50~200 골드 |
| 퀘스트 보상 | 완료 | 100~500 골드 |
| 업적 보석 | 마일스톤 | 10~100 보석 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 현재 점수 (상단)
├── 뱀 길이
├── 일시정지 버튼
└── (타임어택) 남은 시간

레벨 2 (상호작용 시):
├── 게임 오버 점수판
├── 신기록 표시
└── 재시작/메뉴 버튼

레벨 3 (팝업):
├── 신기록 연출
├── 설정 메뉴
└── 상점
```

### 5.2 네비게이션 흐름
```
[메인 메뉴]
    ├── [모드 선택] → [게임] → [게임 오버] → [재시작/메뉴]
    ├── [스킨] → [스킨 선택/구매]
    ├── [상점] → [골드/보석]
    ├── [리더보드] → [랭킹]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 방향 전환 | 부드러운 회전 | - | - |
| 먹이 획득 | 파티클 효과 | 먹이음 | 진동 (약) |
| 길이 증가 | 꼬리 추가 애니메이션 | 성장음 | - |
| 위험 접근 | 경고 표시 | 경고음 | 진동 (약) |
| 충돌 | 폭발 이펙트 | 충돌음 | 진동 (강) |
| 신기록 | 신기록 배너 | 팡파레 | 진동 (중) |

### 5.4 접근성
- **스와이프 조작**: 직관적인 방향 제어
- **버튼 조작**: 화살표 버튼 옵션
- **속도 조절**: 시작 속도 조절 업그레이드
- **색맹 모드**: 뱀/먹이 패턴 구분
- **그리드 표시**: 이동 경로 가이드 옵션
