## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 퍼즐 전투 (블록 매칭)                                │
│         ↓                                               │
│  2. 카드/영웅 스킬 발동                                  │
│         ↓                                               │
│  3. 카드 수집/강화                                       │
│         ↓                                               │
│  4. 덱 구성/시너지 최적화 ─────────────→ (1로 반복)       │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **퍼즐 전투**: 블록 매칭으로 마나/에너지 충전
2. **카드/스킬 발동**: 충전된 에너지로 카드 스킬 사용
3. **카드 수집/강화**: 새 카드 획득, 기존 카드 강화
4. **덱 구성**: 시너지를 고려한 최적 덱 구성

## 3. 재미 축 (Fun Pillars)

### 3.1 빌드/덱 (Build)
- **목표**: 나만의 강력한 덱을 만드는 재미
- **핵심 경험**: 이 조합이 최강이야! 하는 발견
- **주요 요소**:
  - 카드 조합 시스템
  - 시너지 효과
  - 덱 슬롯
  - 테마 빌드

### 3.2 수집 (Collection)
- **목표**: 모든 카드를 모으고 싶은 수집욕
- **핵심 경험**: 새 카드를 얻었을 때의 설렘
- **주요 요소**:
  - 카드 도감
  - 레어도 시스템
  - 한정 카드
  - 수집 보상

### 3.3 퍼즐 판단 (Mastery)
- **목표**: 퍼즐 실력이 결과에 영향을 주는 설계
- **핵심 경험**: 내가 잘 해서 이겼다는 성취감
- **주요 요소**:
  - 콤보 시스템
  - 전략적 매칭
  - 보너스 발동
  - 타이밍 판단

## 4. 디자인 레버 (Design Levers)

### 4.1 블록-스킬 연결
| 레버 | 설명 |
|------|------|
| 색/패턴 연결 | 블록 색/패턴과 카드 스킬 발동을 강하게 연결 |
| 구현 방향 | 빨간 블록 5개 → 화염 영웅 궁극기 충전, 파란 블록 콤보 → 얼음 스킬 강화 |

### 4.2 덱 색 분포 영향
| 레버 | 설명 |
|------|------|
| 플레이 스타일 차별화 | 덱 속성/색 분포가 플레이 스타일(안정/폭발)에 영향을 주도록 설계 |
| 구현 방향 | 단색 덱 → 해당 색 보너스 +50%, 다색 덱 → 유연성 보너스 |

### 4.3 시너지 가이드
| 레버 | 설명 |
|------|------|
| 추천 시너지/빌드 | 카드 도감에서 추천 시너지/빌드 예시 제공 |
| 구현 방향 | 카드 상세 페이지에 "추천 조합: A+B+C", 인기 빌드 랭킹 |

### 4.4 퍼즐 성적 보상 연동
| 레버 | 설명 |
|------|------|
| 콤보/턴 → 보상 | 퍼즐 성적(콤보/남은 턴)이 카드 보상/재료에 영향을 주게 설계 |
| 구현 방향 | 5콤보 이상 → 카드 드롭률 +20%, 10턴 이내 클리어 → 강화석 보너스 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 퍼즐 클리어 | 20회 이상 | 스테이지 완료 로그 |
| 덱 변경 빈도 | 주 3회 이상 | 덱 편집 이벤트 |
| 카드 수집률 | 월 60% 이상 | 도감 달성률 |
| 평균 콤보 수 | 4회 이상 | 퍼즐 로그 분석 |
| D7 리텐션 | 35% 이상 | 7일차 잔존율 |
