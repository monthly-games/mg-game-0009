---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 카드 선택 | 64×90dp | 탭 |
| 카드 배치 | 손패 → 보드 | 드래그 |
| 카드 상세 | 카드 위 | 롱프레스 |
| 카드 교환 | 보드 카드 | 탭 2회 |

### 터치 피드백
- 선택: 카드 상승 + 하이라이트
- 배치 가능: 슬롯 녹색 글로우
- 배치 불가: 슬롯 빨간색 + 흔들림
- 조합 성공: 카드 플립 + 점수 팝업
