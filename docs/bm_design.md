## 3. 상세 BM 설계

### 3.1 카드/영웅 수집 BM

#### 무료 구조
- 튜토리얼 완료 시 기본 카드 덱 지급
- 스테이지 클리어로 카드 조각 획득
- 일일 무료 뽑기 1회
- SR급 15종 이상 무료 수집 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 일반 뽑기 | 10연차 1회 | 프리미엄 재화 2,500 |
| 픽업 배너 | 신규/한정 카드 확률 UP | 동일 |
| 천장 시스템 | 70연차 SSR 확정 | - |
| 선택권 패키지 | 10연차 + SSR 선택권 | $14.99 |
| 카드 스킨 | 카드 외형 변경 | $1.99~$4.99 |

### 3.2 퍼즐/스테이지 BM

#### 무료 구조
- 스태미너 최대 50, 5분당 1 회복
- 일일 무료 충전 1회
- 스테이지 클리어 보상으로 재화 지급

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 스태미너 즉시 충전 | 50 회복 | 프리미엄 재화 80 |
| 스태미너 패스 | 최대 +20, 회복 3분 | $4.99/월 |
| 스킵 티켓 | 클리어한 스테이지 자동 완료 | 이벤트/상점 |
| 하드 모드 입장 | 고급 보상 스테이지 | 프리미엄 재화 50 |

### 3.3 성장/강화 BM

#### 무료 구조
- 퍼즐 클리어로 강화 재료 획득
- 일일 던전에서 카드 경험치 파밍
- 덱 프리셋 1개 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 강화 패키지 | 카드 강화 재료 대량 | $4.99~$9.99 |
| 덱 슬롯 추가 | 덱 프리셋 +3 | $1.99 |
| 레벨 부스트 | 카드 경험치 +50% (7일) | $1.99 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 스태미너, 추가 뽑기, 카드 드롭률 UP |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 스테이지 무과금 클리어 가능
- 퍼즐 실력이 카드 성능보다 중요
- 시너지/덱 구성이 개별 카드 스펙보다 중요

### 과금 압박 최소화
- 캐주얼 유저는 스태미너 부족 느끼지 않는 수준
- 천장 시스템으로 뽑기 좌절 방지
- 광고로 핵심 자원 충분히 획득 가능
